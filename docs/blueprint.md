# **App Name**: SentinelPaper

## Core Features:

- Dynamic Exam Generation: Assemble secure examination papers based on specific blueprints and curriculum standards using high-precision algorithms.
- Forensic Watermarking Engine: Automatically embed invisible, unique identifiers in each generated PDF to track individual document paths and prevent leaks.
- AI-Powered Leak Tracing: An AI tool that reasons through uploaded leaked imagery to identify forensic markers and pinpoint the point of origin or user.
- Natural Language Security Copilot: An interactive Groq-powered tool to query investigation data, audit logs, and center behavior using natural language.
- Integrity QR Verification: Enable field supervisors to instantly verify the authenticity of physical papers through an encrypted QR-code scanning interface.
- Real-time Threat Monitoring: A dashboard utilizing Supabase Realtime to push immediate alerts for anomalous login behaviors or unauthorized distribution.
- Immutable Security Vault: A centralized ledger stored in Supabase capturing a complete chain of custody and encrypted audit trails for all exam interactions.

## Style Guidelines:

- Primary Color: #3163E8 (A modern Electric Blue reflecting trust and industrial precision).
- Background Color: #0B0C10 (A deep, dark Obsidian with a hint of navy to evoke a secure command-center environment).
- Accent Color: #1AC2E5 (A luminous Cyan for focus points, call-to-actions, and security status indicators).
- Headline Font: 'Space Grotesk' (Sans-serif) for a high-tech, computerized, and scientific visual tone. Body Font: 'Inter' (Sans-serif) for clear readability across dense data and audit tables.
- Thin-line, architectural icons with subtle glows to maintain the sophisticated investigative look.
- Information-dense but organized modular grid system reminiscent of an agency-grade cybersecurity terminal.
- Tactile micro-interactions and Framer Motion transitions that provide haptic-like visual feedback for secure operations.
